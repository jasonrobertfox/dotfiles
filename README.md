#Dotfiles

Quick shell and settings installation.

##Installation and Updating

1. Browse to `~/.dotfiles`
2. Execute: `scripts/bootstrap`

##Shell

ZSH configured with a custom prompt, running on iTerm2 with Monokai Theme
